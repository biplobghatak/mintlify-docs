Outpulse supports team collaboration within workspaces. You can invite
colleagues or clients and assign them a role that limits their access
appropriately.

## Roles

| Role | Description |
|------|-------------|
| **Owner** | Full control. Can manage billing, delete the workspace, transfer ownership, and update any role. Only one owner per workspace. |
| **Manager** | Can do everything except billing, ownership transfer, and role updates. Can invite and remove members. |
| **Member** | Can run monitors, build campaigns, draft, edit, send DMs, and update the outbound tracker. Cannot change ICP, tone profile, schedule, or invite others. |

---

## Permissions by role

| Action | Owner | Manager | Member |
|--------|-------|---------|--------|
| View campaigns, leads, outbound tracker, analytics | ✓ | ✓ | ✓ |
| Create and edit monitors | ✓ | ✓ | ✓ |
| Build campaigns, generate drafts, send DMs | ✓ | ✓ | ✓ |
| Edit drafts | ✓ | ✓ | ✓ |
| Update outbound tracker (status, notes) | ✓ | ✓ | ✓ |
| Change ICP and tone profile | ✓ | ✓ | — |
| Update workspace name | ✓ | ✓ | — |
| Connect and disconnect X accounts | ✓ | ✓ | — |
| Invite and remove members | ✓ | ✓ | — |
| Update a member's role | ✓ | — | — |
| View and manage billing & credits | ✓ | — | — |
| Transfer ownership | ✓ | — | — |
| Delete workspace | ✓ | — | — |

Owners can invite Managers and Members. Managers can invite Members only.

---

## Inviting a team member

1. Go to **Settings → Team**
2. Click **Invite member**
3. Enter their email address
4. Select a role (Manager or Member)
5. Click **Send invite**

They'll receive an email with an invitation link. Once they accept and
sign in, they'll have access to the workspace with the role you assigned.

## Changing a member's role

Only the workspace owner can change roles.

1. Go to **Settings → Team**
2. Find the member
3. Click the role dropdown next to their name
4. Select the new role

Changes take effect immediately.

---

## Removing a member

1. Go to **Settings → Team**
2. Find the member
3. Click **Remove**

The member loses access to the workspace immediately. Their past actions
(approved drafts, sent DMs, notes) remain in the workspace.

---

## Transferring ownership

Only an owner can transfer ownership. The new owner must already be a
member of the workspace.

1. Go to **Settings → Team**
2. Click the dropdown next to the target member
3. Select **Transfer ownership**

You'll be demoted to manager after confirming the transfer. The new owner
gains billing access; you keep everything else.
